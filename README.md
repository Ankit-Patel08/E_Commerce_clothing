# My-Ecommerce 🛒

A responsive and interactive e-commerce frontend application built using **HTML, CSS, and JavaScript**.
This project demonstrates frontend development concepts such as dynamic DOM manipulation, responsive UI design, and cart functionality using browser localStorage.

---

# 🚀 Features

## ✅ Responsive Navigation

- Mobile-friendly navbar
- Hamburger menu for smaller devices
- Responsive layout for desktop and mobile

---

## ✅ Dynamic Add to Cart System

- Add products directly from product cards
- Product information extracted dynamically from DOM
- Cart data stored using localStorage

---

## ✅ Persistent Cart Storage

- Cart items remain saved after page refresh
- Shared cart state across multiple pages

---

## ✅ Dynamic Cart Page

- Cart items rendered dynamically using JavaScript
- Displays:
  - Product Image
  - Product Name
  - Product Price
  - Quantity
  - Subtotal

---

## ✅ Quantity Management

- Update quantity dynamically
- Automatic recalculation of subtotal and total

---

## ✅ Remove Item Functionality

- Remove items directly from cart
- Instant UI update after removal

---

## ✅ Cart Count Badge 🛒

- Real-time cart count in navbar
- Updates automatically when items are added

---

## ✅ Improved User Experience

- Toast notification instead of alert()
- Responsive UI for desktop and mobile devices

---

# ⚙️ How It Works

## Add to Cart Flow

1. User clicks cart icon on product card
2. Product information is extracted dynamically
3. Product data is stored in localStorage
4. Cart badge updates automatically
5. Cart page renders items dynamically
